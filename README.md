# 25-2_Software_Engineering_Project
백석대학교 목양관에 있는 시간표를 기반으로 빈 강의실을 확인하여 지도로 띄워주는 프로그램입니다.
백석대학교 소프트웨어공학 25-2 팀프로젝트로 나온 결과물입니다.
역할은 코드에 직/간접적인 기여를 한 것만 표기되어있으며 기타사항의 경우 표기해두지 않았습니다.

## 사용 스택
- 프론트엔드 :
  * HTML, CSS
- 백엔드 :
  * JavaScript
  React Framework를 프로젝트 초기에 사용했었으나, 팀원의 JS 이해도가 떨어져 스택을 낮췄습니다.

## 프로젝트 진행 기간
- 2025/09/22 ~ 2025/12/15

## 팀원 및 역할 
- **노정명**(팀장) :
  * 목양관 구조/도면 및 관련 자료 조사 및 재조사
  * React(JSX)를 활용한 초기 지도 UI 프로토타입 구현 보
  * “빈 강의실 체커”라는 본연의 기능을 수행하기 위한 핵심 (일괄 상태 확인)을 추가
  * 프로그램 수정 마다 검사 및 달성도 확인.
  
- **강요한** :
  * React(JSX)를 활용한 초기 지도 UI 프로토타입 설계 및 구현
  * 스프레드시트 기반 강의 시간표(timeTable) 데이터 구조 설계 및 작성
  * 층별 강의실 구조 파악 및 누락·오류 검증 후 데이터 보완 작업
  * 페이지 동작 흐름을 고려한 소스코드 구조화 및 기본 UI 구성 참여
  * 목양관 내부 구조·도면 자료 조사 및 데이터 매핑 지원
  * 초기 기능 정의, UI 아이디어 제안, 팀 내 코드 이해를 돕기 위한 기능 설명 및 질의응답 참여
  
- **기태훈** :
  * 임재원 팀원이 제작한 UI를 기반하여 최종 메인 화면 제작, 수정
  * 수업 상태 표시 기능 제작
  * 목양관 구조/도면 관련 자료 조사
  
- **백재민** :
  * 팀원의 프로그램 다루어 보는 소프트웨어 평가자
  * 목양관 구조/도면 관련 자료 조사
  
- **오진영** :
  * 소스코드 보조
  * 목양관 구조/도면 관련 자료 조사
  * React(JSX) 소스 코드 작성에 기여
  
- **임재원** :
  * 메인화면, 시간표 선택 창, 지도 등의 프론트엔트쪽 기술 개발
  * 중복호출되는 함수 제거, 코드의 가독성 향상, 유지보수성 향상 등의 소스코드 리팩토링 진행
  * 바이브코딩으로 코드를 제작한 팀원들에게 코드의 로직을 설명하고 이해시킴
  * 시간표가 표기된 스프레드 시트 제작
  * 목양관 구조/도면 관련 자료 조사
